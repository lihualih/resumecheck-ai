# 📄 ResumeCheck.ai — Free AI Resume Analyzer

Get instant AI-powered resume feedback with scoring, keyword analysis, and actionable improvement suggestions. Works 100% in your browser — your data never leaves your device.

**Live Site**: https://resumecheck-ai-two.vercel.app  
**GitHub**: https://github.com/lihualih/resumecheck-ai

---

## ✨ Features

### 7-Dimension Analysis
- **Contact Info** — Checks for email, phone, LinkedIn, GitHub
- **Resume Structure** — Validates sections (Experience, Education, Skills)
- **Content Quality** — Analyzes word count, quantified achievements, dates
- **Action Verbs** — Detects strong/weak verbs, suggests alternatives
- **Keyword Match** — Role-specific keyword analysis for ATS optimization
- **Readability** — Sentence length, structure, formatting
- **Originality** — Detects clichés and generic phrases

### Supported Target Roles
Software Engineer, Frontend/Backend/Fullstack Developer, Data Scientist, Data Analyst, Product Manager, Project Manager, UI/UX Designer, Marketing Manager, Sales, Accountant, HR Manager, Teacher, Nurse, General

### Key Features
- ✅ Instant scoring (0-100) with animated visual feedback
- ✅ Category breakdown with progress bars
- ✅ Strengths and issues identification
- ✅ Keyword analysis (found vs missing for target role)
- ✅ Action verb suggestions
- ✅ Free tier: 3 analyses per day (localStorage tracked)
- ✅ Dark mode support
- ✅ Mobile responsive
- ✅ 100% client-side — no data uploaded

## ⚡ Tech Stack

- **HTML5 / CSS3 / Vanilla JavaScript** — Zero dependencies
- **Rule-based analysis engine** — No external API needed
- **localStorage** — Usage tracking without accounts

## 🚀 How to Use

1. Visit https://resumecheck-ai-two.vercel.app
2. (Optional) Select your target job role
3. Paste your resume text
4. Click "Analyze My Resume"
5. Review your score and suggestions
6. Make improvements and re-analyze
